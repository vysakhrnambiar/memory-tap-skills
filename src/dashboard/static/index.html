Welcome to Memory Tap

Before you begin, please read and understand what this application does.

What Memory Tap Does

Memory Tap connects to a separate, isolated Chrome browser and navigates websites you use — such as YouTube, ChatGPT, and Google Gemini — to collect your conversation history and watch history into a local database on your machine.

How It Works

  • Launches its own Chrome instance with a separate profile — never touches your main browser.
  • You sign into your accounts inside this isolated Chrome, giving Memory Tap access to your session cookies.
  • A Python program controls this Chrome using the Chrome DevTools Protocol (CDP), navigating pages and reading content like a human.
  • Collection runs periodically in the background (every few hours).

Your Data

  • All data stays on your machine — no uploads, no telemetry.
  • Optional RAG chat sends queries to OpenRouter (requires your own API key).

Risks

  • Session access: You are granting a Python program access to your authenticated sessions.
  • Terms of service: Automated browsing may violate some websites' ToS.
  • Local security: Data is stored unencrypted on your filesystem.
  • This software is open source.
