Welcome to Memory Tap

Before you begin, please read and understand what this application does.

What Memory Tap Does

Memory Tap is a personal knowledge collector that runs on your computer. It connects to a separate, isolated Chrome browser (not your main browser) and navigates websites you use — such as YouTube, ChatGPT, and Google Gemini — to collect your conversation history, watch history, and related data into a local database on your machine.

How It Works

  • Memory Tap launches its own Chrome instance with a separate profile — it never touches your main browser, bookmarks, or other data.
  • You will be asked to sign into your accounts (Google, ChatGPT, etc.) inside this isolated Chrome. This gives Memory Tap access to your session cookies and browsing data within that profile.
  • A Python program controls this Chrome browser using the Chrome DevTools Protocol (CDP) — the same technology used by browser developer tools. It navigates pages, scrolls, and reads page content, mimicking human browsing behavior.
  • Collection runs periodically in the background (every few hours by default). The program opens pages, reads your history, and saves it locally.

Your Data

  • All collected data stays on your machine — in a SQLite database stored in your local AppData folder. Nothing is uploaded to any server.
  • No analytics, telemetry, or tracking of any kind is performed.
  • The only network requests Memory Tap makes are: (1) to the websites you authorize it to collect from, and (2) to check for skill updates from our GitHub repository.
  • If you configure the optional RAG chat feature, your search queries are sent to OpenRouter (a third-party LLM API) to generate answers. Your collected data is included as context in those requests. This feature is entirely optional and requires you to provide your own API key.

Risks You Should Understand

  • Session access: By signing into websites inside Memory Tap's Chrome, you are granting a Python program access to your authenticated sessions. This program can read anything you can see when logged into those websites.
  • Account security: While Memory Tap only reads data (it does not post, delete, or modify anything), any program with session access theoretically could. You are trusting this software to behave as described.
  • Third-party websites: Automated browsing may violate the terms of service of some websites. Use at your own discretion.
  • Local security: Your collected data is stored unencrypted on your local filesystem. Anyone with access to your computer can access this data.
  • Open source: This software is open source. You can inspect the code at github.com/vysakhrnambiar/memory-tap-skills to verify its behavior.
